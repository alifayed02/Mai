## Setup Instructions
```git clone https://github.com/alifayed02/Mai```
### Frontend

```cd Mai/Frontend```

```yarn install```

 ### Backend

```cd Mai/Backend```

```pip install -r requirements.txt```

## Running Instructions
### Frontend
```cd Mai/Frontend```

```yarn start```

### Backend
```cd Mai/Backend```

```python main.py```
